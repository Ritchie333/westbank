D39B: Reads In-Game Keyboard Controls (1-2-3).
Used by the routine at CONTROLS_JOYSTICK.
CONTROLS_KEYBOARD D39B LD A,$F7 Read keys 1-2-3-4-5.
D39D IN A,($FE)
CONTROLS_INPUT_1 D39F BIT 0,A If "1" isn't being pressed then jump to CONTROLS_INPUT_2.
D3A1 JR NZ,CONTROLS_INPUT_2
D3A3 LD A,$12 Set A=$12 and return.
D3A5 RET
CONTROLS_INPUT_2 D3A6 BIT 1,A If "2" isn't being pressed then jump to CONTROLS_INPUT_3.
D3A8 JR NZ,CONTROLS_INPUT_3
D3AA LD A,$10 Set A=$10 and return.
D3AC RET
CONTROLS_INPUT_3 D3AD BIT 2,A If "3" isn't being pressed then jump to CONTROLS_INPUT_O_P.
D3AF JR NZ,CONTROLS_INPUT_O_P
D3B1 LD A,$11 Set A=$11 and return.
D3B3 RET
