D5CA: Draw Character In Doorway.
Draws the character frame at the location of one of the three (open) doors. This is direct function, hence is only used when the door is open (it doesn't utilise the door buffer).
D5CA CALL $D5B0 Retrieve the memory location for the character frame from the passed index in A.
D5CD EX DE,HL
D5CE LD B,$58 Sets the height and width.
D5D0 LD C,$06
D5D2 LD A,($CE1A) Retrieves the currently in-focus door from CE1A.
D5D5 LD HL,$4082 Work out which screen position to use for the currently focused door;
A Screen Address
$01 $4082
$02 $408D
$03 $4098
Jump to D5E6 to draw the character and handle the return.
D5D8 DEC A
D5D9 JR Z,$D5E6
D5DB DEC A
D5DC LD HL,$408D
D5DF JR Z,$D5E6
D5E1 LD HL,$4098
D5E4 JR $D5E6
