D307: Action Controls.
Used by the routine at D2CF.
On entry A will contain one of;
Value Bits Door
$10 00010000 Door 2
$11 00010001 Door 3
$12 00010010 Door 1
This is written to D306, and then decoded into writing a $01, $02 or $03 to D300 to represent which door has been shot at. This routine also writes $01 to D305, D303 and D304 to represent the same thing. Before returning, HL is set to the screen location for displaying the shot mask/ shot image in the centre of the door.
