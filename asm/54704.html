D5B0: Character Index.
Returns the 16 bit memory location of a character from a given character index in A.
D5B0 LD H,$00 Using 6800 as a base address (which is the first character), this works out the character frame from a given index passed in A. To give an example with $1A;
  • Loads $1A as the low-order byte; HL=$001A
  • Times by $200; store this in DE=$3400
  • Loads $1A as the low-order byte; HL=$001A
  • Times by $10; HL=$1A0
  • HL+DE=$35A0
  • Add $6800=9DA0
