D7C5: Bowie logic.
Handles all the logic around drawing Bowie and handling his "special" logic.
Bowie state variables/ flags.
BOWIE_GOOD_BAD D7C5 DEFB $01 If this is $00 then Bowie is "good". If this is $01 then he is "bad" (i.e. there's a bomb under his hat).
BOWIE_FRAME_REF D7C6 DEFB $22 The current character frame index reference.
BOWIE_STATE_REF D7C7 DEFB $06 The current character state.
BOWIE_1_TIMER D7C8 DEFB $00 Character timer 1.
BOWIE_2_TIMER D7C9 DEFB $00 Character timer 2.
BOWIE_3_TIMER D7CA DEFB $00 Character timer 3.
BOWIE_4_TIMER D7CB DEFB $00 Character timer 4.
BOWIE_5_TIMER D7CC DEFB $00 Character timer 5.
BOWIE_6_TIMER D7CD DEFB $00 Character timer 6.
This routine looks at the current character state and routes to the correct subroutine.
BOWIE_ROUTING D7CE LD HL,$D7C7 A=BOWIE_STATE_REF.
D7D1 LD A,(HL)
D7D2 DEC A Work out which routine to jump to based on the current state.
A Routine
$01 BOWIE_OPEN_1
$02 BOWIE_OPEN_2
$03 BOWIE_OPEN_3
$04 BOWIE_OPEN_4
$05 BOWIE_CLOSE_2
$06 BOWIE_CLOSE_1
D7D3 JR Z,BOWIE_OPEN_1
D7D5 DEC A
D7D6 JR Z,BOWIE_OPEN_2
D7D8 DEC A
D7D9 JR Z,BOWIE_OPEN_3
D7DB DEC A
D7DC JR Z,BOWIE_OPEN_4
D7DE DEC A
D7DF JR Z,BOWIE_CLOSE_2
D7E1 JR BOWIE_CLOSE_1
Handle door frame 1 countdown/ transition to door frame 2.
BOWIE_OPEN_1 D7E3 LD A,($D7C8) Decrease BOWIE_1_TIMER by one, return if result is not zero.
D7E6 DEC A
D7E7 LD ($D7C8),A
D7EA RET NZ
D7EB INC (HL) Move onto next character state.
D7EC LD A,$1B Pass character index $1B to DRAW_DOOR_FRAME_2 and return.
D7EE CALL DRAW_DOOR_FRAME_2
D7F1 RET
Handle door frame 2 countdown/ transition to door frame 3.
BOWIE_OPEN_2 D7F2 LD A,($D7C9) Decrease BOWIE_2_TIMER by one, return if result is not zero.
D7F5 DEC A
D7F6 LD ($D7C9),A
D7F9 RET NZ
D7FA INC (HL) Move onto next character state.
D7FB LD A,$1B Pass character index $1B to DRAW_DOOR_FRAME_3 and return.
D7FD CALL DRAW_DOOR_FRAME_3
D800 RET
Handle door frame 3 countdown/ transition to door frame 4.
BOWIE_OPEN_3 D801 LD A,($D7CA) Decrease BOWIE_3_TIMER by one, return if result is not zero.
D804 DEC A
D805 LD ($D7CA),A
D808 RET NZ
D809 INC (HL) Move onto next character state.
D80A LD A,$1B Pass character frame index $1B to DRAW_DOOR_FRAME_4 and return.
D80C CALL DRAW_DOOR_FRAME_4
D80F RET
Handle door frame 4 countdown/ transition to door frame 3.
BOWIE_OPEN_4 D810 LD E,(IX+$04) Call HIT_DETECTION with a callback to either BOWIE_CASH_ACTION or BOWIE_BOMB_ACTION.
D813 LD D,(IX+$05)
D816 CALL HIT_DETECTION
D819 LD A,($D7CB) Decrease BOWIE_4_TIMER by one, return if result is not zero.
D81C DEC A
D81D LD ($D7CB),A
D820 RET NZ
BOWIE_CLOSE_3 D821 LD HL,$D7C7 Move onto next character state (BOWIE_STATE_REF).
D824 INC (HL)
D825 LD A,($D7C6) Load the character frame index from BOWIE_FRAME_REF, pass it to DRAW_DOOR_FRAME_3 and return.
D828 CALL DRAW_DOOR_FRAME_3
D82B RET
Handle door frame 3 countdown/ transition to door frame 2.
BOWIE_CLOSE_2 D82C LD A,($D7CC) Decrease BOWIE_5_TIMER by one, return if result is not zero.
D82F DEC A
D830 LD ($D7CC),A
D833 RET NZ
D834 INC (HL) Move onto next character state.
D835 LD A,($D7C6) Load the character frame index from BOWIE_FRAME_REF, pass it to DRAW_DOOR_FRAME_2 and return.
D838 CALL DRAW_DOOR_FRAME_2
D83B RET
Handle door frame 2 countdown/ transition to door frame 1.
BOWIE_CLOSE_1 D83C LD A,($D7CD) Decrease BOWIE_6_TIMER by one, return if result is not zero.
D83F DEC A
D840 LD ($D7CD),A
D843 RET NZ
D844 CALL DRAW_DOOR_FRAME_1 Call DRAW_DOOR_FRAME_1 (i.e. no character/ door is shut).
D847 CALL RESET___ Call RESET___.
D84A RET Return.
Handles marking this as something shootable.
BOWIE_CASH_ACTION D84B LD (IX+$06),$05 Writes $05 to IX+$06 and $01 to IX+$07.
D84F LD (IX+$07),$01
D853 LD A,$01 Writes $01 to IX+$08 and D190.
D855 LD (IX+$08),A
D858 LD ($D190),A
D85B JR BOWIE_CLOSE_3 Jump to BOWIE_CLOSE_3.
Handles marking this as something unshootable.
BOWIE_BOMB_ACTION D85D LD A,$05 Writes $05 to D2FE and return.
D85F LD ($D2FE),A
D862 RET
Handles both shooting the hats off Bowie's head, and checking/ routing on the last frame for the "surprise".
BOWIE_SHOT_ACTION D863 LD HL,$D7C6 A=BOWIE_FRAME_REF.
D866 LD A,(HL)
D867 CP $21 Is it character frame reference (1 hat | $21)?
D869 LD (IX+$06),$01 Writes $01 to IX+$06.
D86D JR NC,BOWIE_CHECK If Bowie is on the last frame then jump forward to BOWIE_CHECK.
D86F INC A Else update BOWIE_FRAME_REF to point at the next frame, call DRAW_DOOR_FRAME_4 and return.
D870 LD (HL),A
D871 CALL DRAW_DOOR_FRAME_4
D874 RET
Checks the "good/ bad" flag and routes the gameplay accordingly.
BOWIE_CHECK D875 LD A,($D7C5) If BOWIE_GOOD_BAD is zero jump to BOWIE_STATE_CASH.
D878 AND A
D879 JR Z,BOWIE_STATE_CASH
D87B JR BOWIE_STATE_BOMB Jump to BOWIE_STATE_BOMB.
Handle displaying cash character frame.
BOWIE_STATE_CASH D87D LD A,$23 Writes character frame reference cash | $23 to BOWIE_FRAME_REF, and pass it to DRAW_DOOR_FRAME_4.
D87F LD ($D7C6),A
D882 CALL DRAW_DOOR_FRAME_4
D885 LD HL,$D84B Writes BOWIE_CASH_ACTION to the current door flags (e.g. DOOR_1_BOWIE_ACTION, DOOR_2_BOWIE_ACTION or DOOR_3_BOWIE_ACTION).
D888 LD (IX+$04),L
D88B LD (IX+$05),H
D88E RET Return.
Handle displaying bomb character frame.
BOWIE_STATE_BOMB D88F LD A,$22 Writes character frame reference bomb | $22 to BOWIE_FRAME_REF, and pass it to DRAW_DOOR_FRAME_4.
D891 LD ($D7C6),A
D894 CALL DRAW_DOOR_FRAME_4
D897 LD HL,$D85D Writes BOWIE_BOMB_ACTION to the current door flags (e.g. DOOR_1_BOWIE_ACTION, DOOR_2_BOWIE_ACTION or DOOR_3_BOWIE_ACTION).
D89A LD (IX+$04),L
D89D LD (IX+$05),H
D8A0 RET Return.
