CDAE: Draw Numbering from Character Set Index.
Used by the routines at CBD0 and CD90.
CDAE PUSH DE Using $C000 as a base address, this works out the character set memory address from a given index passed in A. To give an example using "2" ($25 - see CD90);
  • Loads $25 as the low-order byte; HL=$0025
  • Times by $10; HL=$250
  • Add $C000=C250
This is then passed to D5E6.
CDAF LD H,$00
CDB1 LD L,A
CDB2 ADD HL,HL
CDB3 ADD HL,HL
CDB4 ADD HL,HL
CDB5 ADD HL,HL
CDB6 LD DE,$C000
CDB9 ADD HL,DE
CDBA EX DE,HL
CDBB POP HL
CDBC LD BC,$1001
CDBF CALL $D5E6
CDC2 RET Return.
