D61E: Write Character And Door To Buffer.
Door frames 2 and 3 both contain a "section" of a game character. This is copied into a buffer by first drawing the game character fully to EA90. Then the door frame is drawn over the top (in the correct position).
Draws the game character together with door frame 2.
D61E CALL $D5B0 On entry A points to a character index. The call to D5B0 translates this to the memory location of the character frame data (held in HL) and then calls D654.
